Drawing in Color

This chapter describes the :ink drawing option and the simpler values that can be supplied for that option, such as colors. More complex values that have a regular or irregular pattern in the ink are described in Chapter 14.

SWM We need to add a thing called a ``raster ink'', which includes things like plane masks, pixel values, etc. Be clear that this is a platform and device dependent.

SWM We need to add a thing called a ``palette'', which is simply an abstract color map.
