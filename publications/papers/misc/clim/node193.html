Incremental Redisplay Stream Protocol

stream-redisplayable-p stream

Returns true for any stream that maintains an output history and supports the incremental redisplay protocol, otherwise returns false.

incremental-redisplay stream position erases moves draws erase-overlapping move-overlapping

erases, moves, draws, erase-overlapping, and move-overlapping are differences lists as returned by compute-differences.

incremental-redisplay can be called on any stream that can set the cursor position and replace existing output.
