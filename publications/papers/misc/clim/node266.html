Pane Queries

pane-frame pane

This function returns the frame that ``owns'' the pane. pane-frame can be invoked on any pane in a frame's pane hierarchy, not just its single root pane, but it can only be invoked on ``active'' panes, those that are currently adopted into the frame's pane hierarchy.

pane-foreground pane pane-background pane

Return the current foreground and background inks of the pane, respectively.
