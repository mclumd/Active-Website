Abstract Gadgets

The class push-button we introduced above is an abstract gadget class. The callback interface to all implementations of the gadget is defined by the abstract class. In a frame pane layout specification, a gadget pane is named by its abstract class name. At pane realization time, this abstract class must be resolved to a specific implementation class. The individual implementation classes specify the detailed look and feel of the gadget. Every implementation of an abstract gadget class is a subclass of the abstract class; therefore they all share the same programmer interface. The selection of a specific implementation of an abstract gadget is handled by the frame manager.
