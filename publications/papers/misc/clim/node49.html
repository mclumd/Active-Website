Properties of Sheets

sheet

The protocol class that corresponds to a sheet, a subclass of bounding-rectangle. This and the next chapter describe all of the sheet protocols. a sheet sheet

All of the subclasses of sheet are mutable.

sheetp object

Returns true if object is a sheet, otherwise returns false.
