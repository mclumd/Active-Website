Presentations

A presentation is a special kind of output record that remembers not only output, but the object associated with the output and the semantic type associated with that object.

SWM What exactly is a presentation? What does it mean for presentations to be nested?

presentation

The protocol class that corresponds to a presentation. A subclass of output-record. a presentation presentation

presentationp object

Returns true if object is a presentation, otherwise returns false.

standard-presentation

The output record class that represents presentations. present normally creates output records of this class. Members of this class are mutable.

:object :type :view :single-box :modifier

All presentation classes must handle these five initargs, which are used to specify, respectively, the object, type, view, single-box, and modifier components of a presentation.
