The Pointer Protocol

pointer

The protocol class that corresponds to a pointing device. a pointer pointer Members of this class are mutable.

pointerp object

Returns true if object is a pointer, otherwise returns false.

:port

The :port initarg is used to specify the port with which the pointer is associated.

standard-pointer

The instantiable class that implements a pointer.

pointer-port pointer

Returns the port with which the pointer pointer is associated.

pointer-sheet pointer (setf pointer-sheet) sheet pointer

Returns (or sets) the sheet over which the pointer pointer is located.

pointer-buttons pointer

Returns the state of the buttons of the pointer pointer. This is represented in the same as pointer-event-button.

pointer-position* pointer

Returns the 1#1 and 2#2 position of the pointer pointer as two values.

(setf pointer-position*) x y pointer

Sets the 1#1 and 2#2 position of the pointer pointer to the specified position.

pointer-cursor pointer (setf pointer-cursor) cursor pointer

A pointer object usually has a visible cursor associated with it. These functions return (or set) the cursor associated with the pointer pointer.
