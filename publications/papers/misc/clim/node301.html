Encapsulating Stream Classes

encapsulating-stream

The protocol class that corresponds to an encapsulating stream. an encapsulating stream encapsulating-stream Members of this class are mutable.

encapsulating-stream-p object

Returns true if object is an encapsulating stream, otherwise returns false.

:stream

All encapsulating streams must handle the :stream initarg, which is used to specify the stream to be encapsulated.

standard-encapsulating-stream

This class provides a standard implementation of an encapsulating stream.
