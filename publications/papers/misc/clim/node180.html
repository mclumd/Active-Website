Item List Formatting Protocol

item-list-output-record

The protocol class that represents an item list; a subclass of output-record. an item list output record item-list-output-record

item-list-output-record-p object

Returns true if object is an item list output record, otherwise returns false.

:x-spacing :y-spacing :initial-spacing :n-rows :n-columns :max-width :max-height

All subclasses of item-list-output-record must handle these initargs, which are used to specify, respectively, the 1#1 and 2#2 spacing, the initial spacing, the desired number of rows and columns, and maximum width and height attributes of the item list.

standard-item-list-output-record

The output record that represents item list output. Its children will be a sequence of cells, with presentations possibly intervening. This is a subclass of item-list-output-record.

map-over-item-list-cells function item-list-record

Applies function to all of the cells in item-list-record. map-over-item-list-cells must skip over intervening non-table output record structure, such as presentations. function is a function of one argument, an output record corresponding to a cell in the item list; it has dynamic extent.

adjust-item-list-cells item-list-record

This function is called after the item list output has been collected, but before the record has been replayed. The method on standard-item-list-output-record implements the usual item list layout constraint solver.
