Text Cursor Protocol

Many streams that maintain a text cursor display some visible indication of the text cursor. The object that represents this display is (somewhat confusingly) also called a cursor.

cursor

The protocol class that corresponds to cursors. a cursor cursor Members of this class are mutable.

cursorp object

Returns true if object is a cursor, otherwise returns false.

:sheet

The :sheet initarg is used to specify the sheet with which the cursor is associated.

standard-text-cursor

The instantiable class that implements a text cursor. Typically, ports will further specialize this class.

cursor-sheet cursor

Returns the sheet with which the cursor cursor is associated.

cursor-position* cursor

Returns the 1#1 and 2#2 position of the cursor cursor as two values.

(setf cursor-position*) x y cursor

Sets the 1#1 and 2#2 position of the cursor cursor to the specified position.

cursor-visibility cursor (setf cursor-visibility) visibility cursor

Returns (or sets) the visibility of the cursor cursor. The visibility can be either :on (meaning that the cursor will be visible at its current position), :off (meaning that the cursor is active, but not visible at its current position), or nil (meaning that the cursor is to be deactivated).

display-cursor cursor state

This is responsible for drawing or erasing the cursor cursor. If state is :draw, the cursor will be drawn. Otherwise, if the state is :erase, the cursor will be erased.
