Standard Output

A sheet's output contract is responsible for 1) providing a means of doing output to a sheet and 2) delivering repaints requests to the sheet's client. The standard output contract supports the use of the CLIM output facilities describe in Part . A standard output sheet is provided with its own medium object that can be used to draw on the sheet using the CLIM graphics functions. In addition, these functions can be called directly on the sheet.
