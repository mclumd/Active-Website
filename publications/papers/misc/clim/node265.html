Pane Initialization Options

Client code should be particularly cautious about passing values for these pane options, since the desktop's look and feel or the user's preferences should usually determine these values.

:foreground :background

These options specify the default foreground and background inks for a pane.

:text-style

This option specifies the default text style that should be used for any sort of pane that supports text output. Panes that do not support text output ignore this option.
