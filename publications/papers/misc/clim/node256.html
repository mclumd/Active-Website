Window Viewport Management

The viewport of a window is the area of the window's drawing plane that is currently visible in the window.

SWM Where does this discussion of viewports really belong?

window-clear window

Clears the entire drawing plane by filling it with the background design of the window stream window. If window has an output history, that is cleared as well. The text cursor position of window, if there is one, is reset to the upper left corner.

window-refresh window

Clears the visible part of the drawing plane of window, and then if the window stream is an output recording stream, the output records in the visible part of the window are replayed.

window-viewport window

Returns the viewport of window, which is an object of type area. Usually the viewport will be a standard-rectangle.

window-erase-viewport window

Clears the visible part of the drawing plane of window by filling it with the background design.

window-viewport-position* window

Returns two values, the 1#1 and 2#2 position of the top-left corner of window's viewport.

(setf window-viewport-position*) x y window

Sets the position of window's viewport to x and y.
