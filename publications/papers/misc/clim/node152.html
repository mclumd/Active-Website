Stream Text Cursor Protocol

The following generic functions comprise the stream text cursor protocol. Any extended output stream class must implement methods for these generic functions.

stream-text-cursor stream (setf stream-text-cursor) cursor stream

Returns (or sets) the text cursor object for the stream stream.

stream-cursor-position* stream

Returns the current text cursor position for the extended output stream stream as two integer values, the 1#1 and 2#2 positions.

(setf stream-cursor-position*) x y stream

Sets the text cursor position of the extended output stream stream to x and y. x and y are in device units, and must be integers.

stream-increment-cursor-position* stream dx dy

Moves the text cursor position of the extended output stream stream relatively, adding dx to the 1#1 coordinate and dy to the 2#2 coordinate. Either of dx or dy may be nil, meaning the the 1#1 or 2#2 cursor position will be unaffected. Otherwise, dx and dy must be integers.
