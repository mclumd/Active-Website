Layout Pane Options

:align-x :align-y

The :align-x option is one of :right, :center, or :left. The :align-y option is one of :top, :center, or :bottom. These have the same semantics as for formatting-cell.

:x-spacing :y-spacing :spacing

These spacing options apply to the vbox, hbox, and table panes, and indicate the amount of horizontal and vertical spacing (respectively) to leave between the items in boxes or rows and columns in table. The values of these options are specified in the same way as the :x-spacing and :y-spacing options to formatting-table. :spacing specifies both the 1#1 and 2#2 spacing simultaneously.
